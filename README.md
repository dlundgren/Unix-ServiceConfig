Unix-ServiceConfig
==================

System Services Configuration Interface

INSTALLATION
------------

To install this module type the following:

<pre>
   perl Makefile.PL
   make
   make test
   make install
</pre>

DEPENDENCIES
------------

This module requires these other modules and libraries:

- Term::ReadLine
- Term::ANSIColor
- DBI
- DBD::mysql
- File::Basename
- vpopmail
